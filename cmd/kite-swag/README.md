# SwagX

## 安装

```bash
go install github.com/inhere/kite-go/cmd/kite-swag@latest
```

## 相关库

- https://github.com/getkin/kin-openapi Go 的 OpenAPI 3.0（和 Swagger v2）实现（解析、转换、验证等）
- https://github.com/pb33f/libopenapi 是一个功能全面、高性能的 OpenAPI 3.1、3.0 和 Swagger 解析器、库、验证器和工具包
  - https://github.com/rest-sh/restish
- https://github.com/swaggo/swag 使用 Swagger 2.0 为 Go 自动生成 RESTful API 文档。
- https://github.com/go-swagger/go-swagger Swagger 2.0 实现
